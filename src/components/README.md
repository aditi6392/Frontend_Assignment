# Frontend Assignment – Login & Signup UI

## Description
This project demonstrates a **responsive Login and Signup interface** built using **React** and **CSS**.  
It mimics a real-world authentication flow, including form validations and success screens, without a backend.

---

## Features
- **Login Page**
  - Email & password input
  - Validations: email format, password length
  - Error messages for invalid input
- **Signup Page**
  - Full name, email, password, confirm password
  - Validations: name length, email format, password length, password match
  - Error messages for invalid input
- **Toggle Functionality**
  - Switch between Login ↔ Signup without page reload
- **Success Screens**
  - Mimic Login/Signup success
- **Responsive Design**
  - Works on mobile and desktop devices
- **Clean & Well-Commented Code**
  - Easy to read and maintain
